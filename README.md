# AltruGreen - Golf Charity Subscription Platform

This is a production-grade SaaS platform where golf players can subscribe, submit scores, and participate in charity-driven prize draws.

## Project Structure

- `backend/`: Express.js API with TypeScript and MongoDB.
- `frontend/`: Next.js 14 Web Application.

## Setup

1.  **Backend**:
    - `cd backend`
    - `npm install`
    - Create a `.env` file based on `.env.example`.
    - `npm run dev`

2.  **Frontend**:
    - `cd frontend`
    - `npm install`
    - `npm run dev`

## Deployment

The project is designed to be deployed on Vercel or similar platforms.
